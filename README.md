# texteffect
texteffect for linux shell, color text 
```
from texteffect import *
print(blue('this is blue!'))
print(blueonred('this should be blue on red background!'))
print(blink('this should blink!'))
print(bluebackgroud('this should have blue background!'))
print(reversdisplay('this should swap foreground and background'))
print(highlight("this should be highlighted"))
print(underline("this should be display with underline"))
print(blank("this should be repalced with blank"))
print(diy("this should be dislayed with control code 123", 123))
```
